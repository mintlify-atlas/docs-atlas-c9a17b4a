This page provides a comprehensive reference of all 86 text transformation commands available in sttr.

## Encoding & Decoding

### ASCII85

| Command | Description | Example |
|---------|-------------|-------|
| `ascii85-encode` | Encode text to ASCII85 format | `hello` ã `BOu!rDZ` |
| `ascii85-decode` | Decode ASCII85 to plain text | `BOu!rDZ` ã `hello` |

### Base32

| Command | Description | Aliases |
|---------|-------------|----------|
| `base32-encode` | Encode text to Base32 | `b32-enc`, `b32-encode` |
| `base32-decode` | Decode Base32 to plain text | `b32-dec`, `b32-decode` |
| `crockford-base32-encode` | Encode using Crockford Base32 | - |
| `crockford-base32-decode` | Decode Crockford Base32 | - |

### Base58

| Command | Description | Aliases |
|---------|-------------|----------|
| `base58-encode` | Encode text to Base58 | `b58-enc`, `b58-encode` |
| `base58-decode` | Decode Base58 to plain text | `b58-dec`, `b58-decode` |

### Base62

| Command | Description | Aliases |
|---------|-------------|----------|
| `base62-encode` | Encode text to Base62 | `b62-enc`, `b62-encode` |
| `base62-decode` | Decode Base62 to plain text | `b62-dec`, `b62-decode` |

### Base64

| Command | Description | Aliases | Flags |
|---------|-------------|---------|-------|
| `base64-encode` | Encode text to Base64 | `b64-enc`, `b64-encode` | `-r, --raw` (unpadded) |
| `base64-decode` | Decode Base64 to plain text | `b64-dec`, `b64-decode` | `-r, --raw` (unpadded) |
| `base64url-encode` | Encode text to Base64 URL-safe format | - | - |
| `base64url-decode` | Decode Base64 URL-safe format | - | - |

**Example with flags:**
```bash
echo "hello" | sttr base64-encode
# aGVsbG8K

echo "hello" | sttr base64-encode --raw
# aGVsbG8K (unpadded)
```

### Hexadecimal

| Command | Description | Aliases |

**Example:**
```bash

echo "#FF5733" | sttr hex-rgb

```

## Hashing & Cryptography

### Hash Functions

| Command | Description | Aliases | Output Length |
|---------|-------------|---------|---------------|
| `md5` | Generate MD5 hash | `md5-sum` | 128-bit (32 hex chars) |
| `sha1` | Generate SHA-1 hash | `sha1-sum` | 160-bit (40 hex chars) |
| `sha224` | Generate SHA-224 hash | `sha224-sum` | 224-bit (56 hex chars) |
| `sha256` | Generate SHA-256 hash | `sha256-sum` | 256-bit (64 hex chars) |
| `sha384` | Generate SHA-384 hash | `sha384-sum` | 384-bit (96 hex chars) |
| `sha512` | Generate SHA-512 hash | `sha512-sum` | 512-bit (128 hex chars) |

**Example:**
```bash
echo "password" | sttr sha256
# 5e884898da28047151d0e56f8dc6292773603d0d6aabbdd62a11ef721d1542d8
```

### BLAKE2

| Command | Description |
|---------|-------------|
| `blake2b` | Generate BLAKE2b hash |
| `blake2s` | Generate BLAKE2s hash |

### Checksums

| Command | Description |
|---------|-------------|
| `adler32` | Calculate Adler-32 checksum |
| `crc32` | Calculate CRC-32 checksum |
| `xxh32` | Calculate xxHash 32-bit |
| `xxh64` | Calculate xxHash 64-bit |
| `xxh128` | Calculate xxHash 128-bit |

### Password Hashing

| Command | Description | Flags |
|---------|-------------|-------|
| `bcrypt` | Generate bcrypt hash | `-r, --number-of-rounds` (default: 10) |

**Example:**
```bash
echo "mypassword" | sttr bcrypt
# $2a$10$...

echo "mypassword" | sttr bcrypt --number-of-rounds 12
# $2a$12$...
```

## Text Case Conversion

| Command | Description | Example |
|---------|-------------|-------|
| `lower` | Convert to lowercase | `HELLO` ã `hello` |
| `upper` | Convert to UPPERCASE | `hello` ã `HELLO` |
| `title` | Convert to Title Case | `hello world` ã `Hello World` |
| `camel` | Convert to camelCase | `hello world` ã `helloWorld` |
| `pascal` | Convert to PascalCase | `hello world` ã `HelloWorld` |
| `snake` | Convert to snake_case | `hello world` ã `hello_world` |
| `kebab` | Convert to kebab-case | `hello world` ã `hello-world` |
| `slug` | Convert to URL-friendly slug | `Hello World!` ã `hello-world` |

**Example:**
```bash
echo "user profile settings" | sttr camel
# userProfileSettings

echo "user profile settings" | sttr snake
# user_profile_settings
```

## Line Operations

### Line Manipulation

| Command | Description | Aliases |
|---------|-------------|----------|
| `count-lines` | Count number of lines | - |
| `number-lines` | Prepend line numbers | `nl`, `line-numbers`, `numberlines` |
| `reverse-lines` | Reverse the order of lines | - |
| `shuffle-lines` | Randomly shuffle lines | - |
| `sort-lines` | Sort lines alphabetically | - |
| `unique-lines` | Remove duplicate lines | - |

**Example:**
```bash
cat file.txt | sttr number-lines
# 1. First line
# 2. Second line
# 3. Third line

cat file.txt | sttr unique-lines
# Returns only unique lines in original order
```

## Text Manipulation

### Basic Operations

| Command | Description | Example |
|---------|-------------|-------|
| `reverse` | Reverse the entire text | `hello` ã `olleh` |
| `remove-newlines` | Remove all newline characters | - |
| `remove-spaces` | Remove all spaces | `hello world` ã `helloworld` |

### Quote Handling

| Command | Description | Aliases | Flags |
|---------|-------------|---------|-------|
| `escape-quotes` | Escape quotes in text | `esc-quotes` | `-d` (double), `-s` (single) |

**Example:**
```bash
echo 'He said "hello"' | sttr escape-quotes
# He said \"hello\"

echo 'He said "hello"' | sttr escape-quotes --double-quote
# He said \"hello\"
```

### Zero Padding

| Command | Description | Flags |
|---------|-------------|-------|
| `zeropad` | Pad numbers with zeros | `-n, --number-of-zeros` (default: 5)<br/>`-p, --prefix` (prefix string) |

**Example:**
```bash
echo "42" | sttr zeropad
# 0000042

echo "42" | sttr zeropad -n 3
# 00042

echo "42" | sttr zeropad -n 3 -p "ID_"
# ID_00042

echo "-42" | sttr zeropad -n 3
# -00042
```

## Finding Commands

You can find commands in several ways:

1. **Interactive mode**: Run `sttr` without arguments to browse all processors
2. **List all**: Use `sttr --help` to see all available commands
3. **Search by alias**: Many commands have convenient aliases (e.g., `b64-enc` for `base64-encode`)
