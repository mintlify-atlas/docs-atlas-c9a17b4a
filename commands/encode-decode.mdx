sttr supports a wide range of encoding and decoding formats for text transformation.

## Base64 Encoding

<CardGroup cols={2}>
  <Card title="base64-encode" icon="arrow-up-right-from-square">
    Encode text to Base64 format
  </Card>
  <Card title="base64-decode" icon="arrow-down-left-to-square">
    Decode Base64 text to plain text
  </Card>
</CardGroup>

### Usage

```bash
# Encode to Base64
echo "hello world" | sttr base64-encode
# Output: aGVsbG8gd29ybGQ=

# Decode from Base64
echo "aGVsbG8gd29ybGQ=" | sttr base64-decode
# Output: hello world

# Raw (unpadded) encoding
echo "hello" | sttr base64-encode -r
# Output: aGVsbG8
```

**Aliases:** `b64-enc`, `b64-encode`, `b64-dec`, `b64-decode`

**Flags:**
- `-r, --raw` - Use unpadded Base64 encoding

## Morse Code

<CardGroup cols={2}>
  <Card title="morse-encode" icon="signal">
    Encode text to Morse code
  </Card>
  <Card title="morse-decode" icon="signal">
    Decode Morse code to text
  </Card>
</CardGroup>

### Usage

```bash
# Encode to Morse code
echo "SOS" | sttr morse-encode
# Output: ... --- ...

echo "HELLO" | sttr morse-encode
# Output: .... . .-.. .-.. ---

# Decode from Morse code (default: Latin)
echo "... --- ..." | sttr morse-decode
# Output: SOS

# Decode with specific language
echo "... --- ..." | sttr morse-decode -l ru
# Output: (Cyrillic equivalent)
```

**Aliases:** `morse-enc`, `morse-code-encode`, `morse-code-enc`, `morse-dec`, `morse-code-decode`, `morse-code-dec`

**Flags:**
- `-l, --lang` - Morse code language set (la, ru, gr, he, ar, ja, kr, th)
  - `la` - Latin (default)
  - `ru` - Cyrillic
  - `gr` - Greek
  - `he` - Hebrew
  - `ar` - Arabic
  - `ja` - Japanese
  - `kr` - Korean
  - `th` - Thai

**Note:** Morse code uses space to separate letters and `/` to separate words.

## ROT13

<Card title="rot13" icon="rotate">
  Apply ROT13 cipher (both encode and decode)
</Card>

### Usage

```bash
# ROT13 encode (also decodes when applied twice)
echo "hello" | sttr rot13
# Output: uryyb

# ROT13 decode (same command)
echo "uryyb" | sttr rot13
# Output: hello

# Preserve case
echo "Hello World" | sttr rot13
# Output: Uryyb Jbeyq
```

**Aliases:** `rot13-encode`, `rot13-enc`

**Note:** ROT13 is symmetric - applying it twice returns the original text. It only affects letters (A-Z, a-z) and preserves numbers, punctuation, and case.

## Use Cases

### API Development
```bash
# Encode credentials for Basic Auth
echo -n "username:password" | sttr base64-encode
# Output: dXNlcm5hbWU6cGFzc3dvcmQ=
```

### Web Development
```bash
# URL-encode query parameters
echo "search query" | sttr url-encode
# Output: search+query

# Escape HTML in user input
echo "<script>alert('xss')</script>" | sttr html-encode
# Output: &lt;script&gt;alert(&#39;xss&#39;)&lt;/script&gt;
```

### Data Processing
```bash
# Convert binary data to hex for inspection
cat binary.dat | sttr hex-encode | head -c 100

# Decode Base64-encoded configuration
cat config.b64 | sttr base64-decode
```

### Fun & Education
```bash
# Send secret messages
echo "meet at midnight" | sttr morse-encode
# Output: -- . . - / .- - / -- .. -.. -. .. --. .... -

# Simple text obfuscation
echo "secret" | sttr rot13
# Output: frperg
```
